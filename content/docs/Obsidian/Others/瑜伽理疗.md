# 颈椎曲度变直
- 原因
	- 当胸椎曲度变直了，颈椎曲度就容易变直
	- 经常低头
	- 头前移，侧面的体态，可能是打电脑打出来的
- 后果
	- 它会导致到我们的上颈椎的肌肉变比较紧，靠近我们头骨的区域变紧，然后颈椎的下段的肌肉变得无力
	- 可能会形成一些手臂的麻木，手臂的循环，颈椎的疼痛，呼吸模式
